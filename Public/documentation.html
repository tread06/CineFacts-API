CineFacts API Documentation

The CineFacts API provides information on movies and directors and allows users to create profiles and shareable movie collections.

Create a new user from JSON data.

POST
https://cinefacts-api.herokuapp.com/users

Request Body {
 "Username": "Jeb",
 "Password": "password1234",
 "Email": "Jeb@test.com",
 "Birthday": "2000-01-01"
}
Responce Body Returns the created user data.

{
 "UserName": "Jeb",
 "Password": "hashedPassword",
 "Email": "Jeb@test.com",
 "Birthday": "2000-01-01T00:00:00.000Z",
 "FavoriteMovies": [],
 "_id": "6132e637733afaf8560f2d69",
 "__v": 0
}


Updates user data.

PUT
https://cinefacts-api.herokuapp.com/users/:username Requires Authorization

Request Body {
 Username: "Jeb", (required)
 Password: "password1234", (required)
 Email: Jeb@test.com", (required)
 Birthday: 2000-01-01
}
Responce Body Returns the updated user data.

{
 "UserName": "Jeb",
 "Password": "hashedPassword",
 "Email": "Jeb@test.com",
 "Birthday": "2000-01-01T00:00:00.000Z",
 "FavoriteMovies": [],
 "_id": "6132e637733afaf8560f2d69",
 "__v": 0
}


Delete user data.

DELETE
https://cinefacts-api.herokuapp.com/users/:username Requires Authorization

Request Body none
Responce Body TEXT - [User] was deleted.
