CineFacts API documentation

The CineFacts API provides information on movie genres and directors and allows users to create profiles and shareable movie collections.

Users

METHOD Endpoint URL/params Description Request body example Response body example
POST /users Creates a new user {"name": "name"} Text: "user created"
PUT /users/{user}/{name} Updates the {user}'s' name to be {name} none Text: "user name was changed"
POST /users/{user}/movies Adds a movie to the {user}'s movie list using movie data in the request body {title: "title", description: "description", genre: "genre", director: "director", imageUrl: "imageUrl"} Text: "Book added to user's book list."
DELETE /users/{user}/movies/{index} Removes a movie to the {user}'s movie list at {index} none Text: "Book at index was removed from user's book list."

Movies

METHOD Endpoint URL/params Description Request body example Response body example
GET /movies Returns a list of all movies none JSON: {movies: [{title: "title", description: "description", genre: "genre", director: "director", imageUrl: "imageUrl"}]}
GET /movies/{title} Returns movie data using movie {title} none JSON: {title: "title", description: "description", genre: "genre", director: "director", imageUrl: "imageUrl"}
GET /directors/{name} Returns director data using director {name} none JSON: {name: "name", bio: "bio", birthYear: "birthYear", deathYear: "deathYear"}
GET /genres/{name} Returns genre data using genre {name} none JSON: {name: "name", description: "description"}
